Flexbox

Flexbox is a relatively new tool that has made layout design much more accessble for new and experienced developers alike. It allows us to size and resize elements of a web page dynamically, without having to declare strict sizes that don't scale well across many different devices. It is an invaluable tool for responsive designs and really a cornerstone of modern web design. We will cover the basics of flexbox as well as give some examples of how and where to use it!
